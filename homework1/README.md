# 微積分基本定理（數值驗證）

## 一、目標

利用 Python 程式，**以數值方法驗證微積分基本定理（第一基本定理）**：

\[
\frac{d}{dx}\left(\int_0^x f(t)\,dt\right) = f(x)
\]

若先對函數 \(f(t)\) 從 0 積分到 x，再對 x 微分，理論上應該回到原函數 \(f(x)\)。

---

## 二、解題想法

由於電腦無法直接進行解析微分與解析積分，本作業採用：

- **數值微分** 近似導數
- **數值積分** 近似定積分

並比較兩者的結果是否接近，以此驗證微積分基本定理的正確性。

---

## 三、使用的方法說明

### 1. 數值微分（Central Difference）

使用中央差分法近似導數：

\[
f'(x) \approx \frac{f(x+h) - f(x-h)}{2h}
\]

def df(f, x, h=1e-6):
    return (f(x + h) - f(x - h)) / (2 * h)
