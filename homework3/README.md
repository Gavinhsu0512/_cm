# 三次多項式求根（root3）

## 一、目標

使用 Python 撰寫一個函數 `root3(a, b, c, d)`，求解三次多項式：

\[
f(x) = ax^3 + bx^2 + cx + d
\]

程式需滿足以下條件：

- 正確求出三個根
- 即使沒有實根，也能回傳複數根
- 將求得的根代回原多項式驗證，結果應非常接近 0

---

## 二、解題背景與想法

三次多項式在數學上是**最後一個存在通用解析公式的多項式階數**。  
因此，本作業採用三次方程式的經典解法（Cardano 方法），先將原式轉換成標準形式，再利用代數運算求解。

由於計算過程中可能出現負數平方根或複數，因此使用 Python 的 `cmath` 模組來處理複數運算。

---

## 三、數學方法說明

### 1. 將三次多項式轉為降冪形式

原式：
\[
ax^3 + bx^2 + cx + d = 0
\]

令：
\[
x = y - \frac{b}{3a}
\]

可化為：
\[
y^3 + py + q = 0
\]

其中：
\[
p = \frac{3ac - b^2}{3a^2}
\quad
q = \frac{2b^3 - 9abc + 27a^2 d}{27a^3}
\]

---

### 2. 判別式

\[
\Delta = \left(\frac{q}{2}\right)^2 + \left(\frac{p}{3}\right)^3
\]

判別式可能為正、零或負，因此必須使用複數平方根。

---

### 3. Cardano 解法

利用：
\[
A = \sqrt[3]{- \frac{q}{2} + \sqrt{\Delta}}, \quad
B = \sqrt[3]{- \frac{q}{2} - \sqrt{\Delta}}
\]

即可得到三個 y 的解，進而還原出三個 x 的解。
