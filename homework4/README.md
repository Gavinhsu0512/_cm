# 一般 n 次多項式求根（root(c)）

## 一、目標

目標是撰寫一個函數 `root(c)`，用來求解一般 n 次多項式的所有根：

\[
P(x) = c_0 + c_1 x + c_2 x^2 + \cdots + c_n x^n
\]

其中：
- `c[i]` 代表 \(x^i\) 的係數
- 次數 n ≥ 1
- 當 n ≥ 5 時，數學上已證明不存在通用解析公式（伽羅瓦定理）

因此，本作業**不使用公式解**，而是採用**數值方法**近似求出所有（包含複數）根。

---

## 二、理論背景（為何不能用公式）

根據 **伽羅瓦定理（Galois Theory）**：

- 一次到四次多項式：存在通用解析公式
- **五次以上的一般多項式（n ≥ 5）**：不存在能用有限次加減乘除與開根號表示的公式

因此，對於一般 n 次多項式，實務上必須使用**數值演算法**來求根。

---

## 三、解題策略選擇

### 為何使用數值法？

- 不依賴解析公式
- 可同時處理實根與複數根
- 對任意次數 n 都適用
- 為科學計算與工程常用方法

---

## 四、採用的方法：Durand–Kerner（Weierstrass）法

### 方法概念

Durand–Kerner 法是一種**同時求出所有根**的數值迭代法。  
假設多項式有 n 個根，維護 n 個複數估計值 \(z_1, z_2, \ldots, z_n\)，反覆進行以下更新：

\[
z_i \leftarrow z_i - \frac{P(z_i)}{\prod_{j \neq i} (z_i - z_j)}
\]

直到所有根都收斂為止。

---

### 為什麼選這個方法？

- 不需要導數（比 Newton 法穩定）
- 天然支援複數
- 可一次收斂到全部根
- 特別適合 n ≥ 5 的情況
