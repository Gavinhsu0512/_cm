### 2. 資訊理論基本量（Information Measures）

本專案以 **自然對數 ln** 為底（單位：nats）。

#### (1) 熵（Entropy）
\[
H(p) = -\sum_i p_i \log p_i
\]

#### (2) 交叉熵（Cross Entropy）
\[
H(p,q) = -\sum_i p_i \log q_i
\]

#### (3) KL 散度（Kullback–Leibler Divergence）
\[
KL(p \Vert q) = \sum_i p_i \log \frac{p_i}{q_i}
\]

#### (4) 互資訊（Mutual Information）
\[
I(X;Y) = \sum_{x,y} p(x,y)\log \frac{p(x,y)}{p(x)p(y)}
\]

---

### 3. 交叉熵不等式驗證

理論關係式：
\[
H(p,q) = H(p,p) + KL(p\Vert q)
\]

由於：
\[
KL(p\Vert q) \ge 0
\]

可得重要結論：
\[
\boxed{H(p,p) \le H(p,q)}
\]
且當 \( q \neq p \) 時為嚴格不等式。

本專案以程式實際計算並驗證上述關係。

## 二、理論說明

### 1. 夏農信道編碼定理（Shannon Channel Coding Theorem）

此定理說明「可靠通訊的極限」：

- 若傳輸速率 \( R < C \)（信道容量）  
  → 存在編碼方式，使錯誤率可趨近於 0  
- 若 \( R > C \)  
  → 不論使用任何編碼，錯誤率皆無法趨近於 0  

此定理是 **所有錯誤更正碼與通訊系統設計的理論基礎**。

---

### 2. 夏農–哈特利定理（Shannon–Hartley Theorem）

針對 **加性白高斯雜訊（AWGN）信道**，給出信道容量的實際公式：

\[
C = B \log_2(1 + S/N)
\]

其中：
- \( C \)：信道容量（bits/s）
- \( B \)：頻寬（Hz）
- \( S/N \)：訊噪比（線性尺度）

此公式量化了：
- 增加頻寬
- 提升訊噪比  

對通訊容量的影響。

---

### 3. 兩個定理的關係

| 定理 | 角色 |
|----|----|
| 夏農信道編碼定理 | 抽象理論，定義「容量」與可否可靠傳輸 |
| 夏農–哈特利定理 | 在 AWGN 條件下，給出容量的具體數學公式 |
